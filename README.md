# Korean2Num

한글로 된 숫자를 실제 숫자로 변경해주는 툴 입니다

Changing Korean number to natural number

## Usage

```python

import korean2num

# kr_str = input("Number in Korean: ")
kr_str = "일조삼천육백칠십일억이천구백삼십칠만사천오백십일"
val = korean2num.kr2num(kr_str)

print(val)
# >>> 1367129374511

```
